# Cole's Mobile Detailing v6.1 — AI Assistant + Lead Alerts

## Added

- Premium floating website assistant
- Server-side OpenAI Responses API integration
- Safe deterministic quote engine
- Human handoff workflow
- Twilio SMS owner notifications
- Digital Forge lead creation from chat
- Conversation transcript saved to lead notes
- Google Ads and UTM attribution for assistant leads
- CRM-connected quote form at `/book`
- Guided fallback when no AI key is configured
- Environment template and setup documentation

## Important limitations

- Real AI replies require an OpenAI API key.
- Real SMS alerts require a Twilio account, number or Messaging Service, and credentials.
- The assistant gives starting estimates only. Cole confirms final pricing and availability.
- Photo uploads and two-way SMS replies are planned for a future release.
