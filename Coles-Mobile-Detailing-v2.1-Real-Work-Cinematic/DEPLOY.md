# Cole's Mobile Detailing v2.1

## Deploy
1. Extract the ZIP.
2. Upload everything inside the extracted folder to the repository root.
3. Confirm `public/models/porsche/scene-optimized.glb` and `public/videos/` are present in GitHub.
4. Netlify build command: `npm run build`
5. Netlify publish directory: `.next`

## New media
